
# Judul
Prediksi Harga Beras dan Minyak Goreng Menggunakan Metode Linear Regression

## Metode
1. Pengumpulan Data
2. Preprocessing
3. Training Linear Regression
4. Evaluasi MAE, MSE, RMSE, R²
5. Prediksi harga periode berikutnya

## Variabel
X = Bulan ke-1 s.d. 24
Y = Harga Beras / Harga Minyak Goreng
